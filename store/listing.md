# Chrome / Edge Web Store Listing

## Name
AppRoom Customiser

## Short description (≤132 characters)
Adds configurable workflow helpers to the App-Room ERP: print buttons, barcode check-in, registration automation and more.

## Detailed description

AppRoom Customiser extends the App-Room ERP (erp.app-room.ch) with a set of configurable workflow helpers that reduce repetitive clicks and manual data entry. Every feature can be toggled individually from the extension settings page.

**Rental — print current view**
Adds a "Print" action to each rental row. Opens a clean print preview with the visible row details — no need to navigate away or copy data manually.

**Rental — Unterschrift reminder**
Highlights the signature button with a pulsing red animation so it is never overlooked during a rental handover.

**Customer registration → open rental**
Adds duration buttons (Half day / 1 day / 2 days) next to the registration submit button. After the customer is registered, the extension automatically navigates to a new rental, pre-selects the customer, and sets the chosen duration.

**Barcode check-in**
Adds a barcode input field to the storage order view. Scanning an article barcode increments the corresponding check-in quantity immediately.

**Check-in quantity warning**
Colour-codes storage order rows based on check-in status so shortfalls are visible at a glance before confirming the order.

**Label printing by check-in quantity**
Pre-fills the label print dialog with the current check-in quantities, removing the need to enter them manually.

**Registration field customisation**
A per-field matrix lets you move fields into a collapsible "Extra" section, mark fields as mandatory, and override field labels in German, English, Italian and French.

---

All settings are stored locally in your browser. No data is collected, transmitted, or shared.
